# 🎵 SpotiFree

SpotiFree is a modern music player inspired by Spotify, built using HTML, CSS, and JavaScript. It offers a clean interface with smooth playback controls and a responsive design.

## ✨ Features

- 🎵 Music Playback
- ⏯️ Play / Pause / Share / Download
- 🔽 Upload
- ➕ Create Playlist 
- ⏭️ Next & Previous Track
- 📊 Progress Bar
- 🔊 Volume Control
- 📱 Responsive Design
- 🎨 Spotify-Inspired UI

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 🚀 Getting Started

1. Clone this repository
2. Open the project folder
3. Run using Live Server or open `index.html`
